Appium BDD Mobile Testing Framework
This repository contains an automated testing framework for mobile applications using Appium, Cucumber (BDD), Selenium, and TestNG.

Tools & Technologies:
- Java
- Selenium
- Appium Java Client
- Cucumber (BDD)
- TestNG
- Maven
- Android Emulator / Real Device

Framework Features:
- BDD structure with.feature files
- Step Definitions for each scenario
- Page Object Model (POM) design
- Appium driver initialization
- TestNG integration for execution
- Maven for dependency management

 How to Run:
1. Clone the repo:
git clone https://github.com/rams94000/Cielo365MobileAppProject.git

markdown
2. Open in Eclipse or IntelliJ.
3. Start Appium Server.
4. Run your feature file or TestNG runner.
